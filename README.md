# Django Todo API

## Como usar

1. Execute: start.bat
2. Aguarde 30 segundos
3. Acesse: http://localhost:8000/api/

## Endpoints

- POST /api/auth/register/ - Registrar
- POST /api/auth/login/ - Login  
- GET /api/todos/ - Listar tarefas
- POST /api/todos/ - Criar tarefa

## Admin

- URL: http://localhost:8000/admin/
- Usuario: admin
- Senha: admin123

## Parar

Execute: stop.bat
